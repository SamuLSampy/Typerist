
# Typerist

Um projeto para colocar em prática meus avanços no curso de desenvolvimento Web.

### Sobre

Typerist foca principalmente em treinar habilidades de digitação do usuário de forma divertida, com pontuação, rank e level.

### Fase atual

O Projeto está em fase inicial no momento, apenas com o conceito criado.
Tenho essas metas:

 - Backend com Node, express, mongoDB;
 - Placar global e local;
 - Multiplicadores de pontos, como efeito Cambridge;
 - Efeitos visuais aprimorados;
 - Multijogador;

O objetivo é transformar o treino em algo divertido e viciante ao mesmo tempo em que aprimora a digitação do usuário.

Atualizarei o repositório conforme vou aprendendo.
